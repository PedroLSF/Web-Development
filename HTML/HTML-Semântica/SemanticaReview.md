# Review
Parabéns por concluir esta lição sobre HTML semântico! Agora que você conhece os benefícios do HTML semântico e como usá-lo, você pode incorporar elementos semânticos em seu site para torná-lo mais acessível e facilitar a leitura do código.

Vamos rever alguns dos tópicos que abordamos ao longo da lição:

* O HTML semântico introduz significado a uma página por meio de elementos específicos que fornecem contexto sobre o que está entre as tags.
* O HTML semântico é um padrão moderno e torna um site acessível para pessoas que usam leitores de tela para traduzir a página da web e melhorar o SEO do seu site.
* "header", "nav" , "main" e "footer" criam a estrutura básica da página web.
* "section" define elementos em um documento, como capítulos, títulos ou qualquer outra área do documento com o mesmo tema.
* "article" contém conteúdo que faz sentido por conta própria, como artigos, blogs, comentários etc.
* "aside" contém informações relacionadas ao conteúdo principal, mas não necessárias para entender as informações dominantes.
* "figure" encapsula todos os tipos de mídia.
* "figcaption" é usado para descrever a mídia em "figure".
* Os elementos "video", "embed" e "audio" são usados ​​para arquivos de mídia.

Agora, aplique esse conhecimento para se tornar um Desenvolvedor Web melhor.
